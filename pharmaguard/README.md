# PharmaGuard

A pharmacogenomic risk prediction web app.

## Project Structure
- `app.py`: Flask application entry point.
- `parser/`: VCF parsing logic.
- `engine/`: Pharmacogenomic rules (phenotype and drug interactions).
- `llm/`: LLM integration for explanations.
- `templates/`: HTML templates.
- `static/`: CSS and static assets.
